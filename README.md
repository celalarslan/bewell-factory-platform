# Bewell Factory Systems MVP

Interactive Türkiye–Africa modular factory platform prototype.

## Included
- Premium public landing page
- Modular factory catalogue
- Interactive project configurator
- Dynamic investment, land, delivery and employment estimates
- Project readiness score
- Project brief generation state
- Client project portal preview
- Responsive mobile layout

## Run locally
```bash
npm install
npm run dev
```

## Production build
```bash
npm run build
```

This MVP uses indicative demo values only. Commercial estimates must later be connected to verified supplier and country datasets.

## AI Company Operating System

The MVP now includes an interactive AI Project Office layer:

- Celal Arslan executive command center and human approval gate
- Chief of Staff / Orchestrator
- Technical Director
- PMO
- Procurement & Supplier Manager
- Finance & Investment Structuring
- Contract & Compliance
- Proposal & Cost Engineering
- Digital Operations / CRM
- Independent Review / Red Team
- Decision Queue with approve / evidence / reject actions
- Shared Project Dossier and evidence register
- Agent charter modal showing authority and hard boundaries

This release is a front-end operating model. Real agents, authentication, database persistence, workflow execution and external tools are the next implementation phase.
